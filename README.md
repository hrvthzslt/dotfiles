## install

### Setup Nix Env and Install packages

```shell
./setup/nix.sh
```

### Link dotfiles and personal scripts

```shell
./setup/link.sh
```

### Setup keyboard layout (Only for Linux)

```shell
./setup/keyd.sh
```

### Install Kitty terminal

```shell
./setup/kitty.sh
```

### Run ansible playbook

```shell
ansible-playbook -i hosts main.yml
```
